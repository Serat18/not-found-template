# not-found-template
An error page template built from nginx 404 error page.
## Usage
Replace the placeholders in `index.html`:
- `Error name` — your error name in page name e.g. `404 Not Found`
- `[Error here]` — your error name e.g. `404 Not Found`
- `[Service name, version and host here]` — your service info e.g. `nginx/1.25.3 (Ubuntu)`but version and host isn't strictly required.
